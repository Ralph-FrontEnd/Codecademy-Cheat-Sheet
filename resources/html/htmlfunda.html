This page is not designed as a tutorial, but is designed to be informative on the select topics that have been created! Structure
When you're starting your HTML document there is a correct way to setup your page! Often IDE's such as Visual Studio Code have shortcuts to help you, but overall your page setup should have the following structure for you to work with!
Attribute Definition Example
<!DOCTYPE html> The DOCTYPE attribute is referring to HTML 5, this is the latest build of HTML and is the standard to expect on modern day websites. The <!DOCTYPE html> is telling the browser to expect HTML code for the file being loaded.
<head>
</head>		 The <head> attribute will be the first section on your HTML code, this section will contain all of your metadata. Most metadata won't appear on your finished website as text, but contains important infromation to have your website function. Such as linking our CSS Stylesheet!
<title>
</title>		 Our title is important, the title sits in our head section and is exactly as it sounds. The title of your page, this title then appears as text in your browser on your tab/window.
<header>
</header>		 After our Head section you'll then use a <header>, this is the start of your page content and you'll use the header for the likes of a <h1> title and navigation bar.
<body>
</body>		 The main part of our website, just like you as a person, is the body. The <body> of the website contains all your main content, from paragraphs to images and videos!
<footer>
</footer>		 Quite often you'll see at the bottom of websites a contact us area, or something similar. This is the footer, and where you'd store all your miscellaneous links and anything else you want to "sign off" your document with!
Headings
HTML has a lot of built in features, as you'd expect with the many years of development and improvements behind it! Having titles/subtitles within your page are great for readability, and this is where our headings come into play! (Not to be confused with the header section!)
Attribute Definition Example
<h1></h1>
<h2></h2>
<h3></h3>
<h4></h4>
<h5></h5>
<h6></h6>
 Built in to HTML there is a total of 6 headings that you can use, starting with <h1> being the largest heading and <h6> being the smallest heading.
Really, you should only use the <h1> tag once within each HTML page, as you'd want to consider it a "title" of sorts. On the right hand side is an example of how they look without styling!
Classes & ID's
As your code starts to become more advanced, and you're looking to style certain areas using CSS, it's handy to be able to "label" things within your HTML. There is a few ways to do this, but the two we are covering here are classes and ID's.
Attribute Definition Example
class=" " The class could be considered a property value, it's the first part of the declaration that signifies what visual characteristics of the element is to be modified.
Although a class and ID work the same, they do have some slight differences. An important part of classes is they are great to be used multiple times unlike an ID.
A class is used as an extra label on an attribute, a label that you can reference in CSS to style that specific attribute.
id=" " The ID can also be considered a property value just like a class. But an ID has some extra powers, such as overwriting styling on a class, there is a hierarchy when it comes to using these property values.

  • ID's
  • Classes
  • Attribute
  • * = Global modifier

The global modifier has the least power, while an ID has the most power to overwrite styling.
Text Elements
At this point you'll likely be using CSS to style your text, but it's always worth mentioning that there are some text elements that will style your text built into HTML. On top of this there are some text elements such as <a> (anchor tags) that will allow you to use links!
Attribute Definition Example
<strong>
</strong> 		The <strong> tag will change the text within the opening and closing tags to be bold.
So the example on the right, the text tectonic plates will appear bold!
<em>
</em>		 The <em> tag stands for emphasis, and this will change any text within the opening/closing tags to itallics!
So for example, the text in the example, the (go underneath another) would appear in itallics!
<a>
</a>		 The <a> tag is known as an anchor tag, and the anchor tag allows you to add links to stuff. From text, to images even videos. Whatever you wrap inside of the anchor tag will become a clickable link.

When using an anchor tag you'll put the link inside of href="" which goes on the opening tag. Then between the opening tag and closing tag you will put your image, or text, or video that you want hyperlinked! Just as shown in the example!
<br> When writing text in your paragraph, on your code editor of choice, if you take your text to a new line it won't make a new line on the web page. Just continue on from where you left off!

To get spaces whilst staying within the same element, like I've just done here, you use <br> which has no closing tag!
Image's & Videos
A huge part of making your website feel welcoming can be videos and images, this section will cover how you can add images and videos, and some of the modifications you can make too!
Attribute Definition Example
<img/> The <img> tag doesn't require a closing tag, the reason for the added / allows for the tag to be self-closing!

The tag will mainly request 2 pieces of information.
The src" "
The alt" "

The SRC will be the location of your image, this can be a page URL, or linking to the image within your local files.

The ALT stands for alternate, and is text you will want to represent your image. It won't technically be visible, unless the image fails to load or a screen reader is being used.
<video>
</video>		 Unlike the img tag the <video> does require an opening & closing tag!

To make the video tag function, it just needs the link of a video! You are also able to add text in between the >< tags. This text will work the same as the ALT text on image!
Creating Lists
Lists are an important part of HTML, creating some basic structure for contents to be kept together. Whether it be a list of locations, a list of priorities, to outline your navigation items, or ways to contact you through the footer. Lists have a number of possibilities and here I'll show you how to create an ordered list, and an unordered list!
Attribute Definition Example
<ul>
</ul>		 The <ul> tag is the start to an Unordered List! It's the parent of the <li> </li> tags, and we use the <ul> tag to specify the type of list we are using!

An unordered list uses bulletpoints instead of numbers, and for each item you want in the list, you add a new set of <li> </li> tags.
<ol>
</ol>		 The Ordered List works exactly the same way as our unordered list, the <ol> is the parent and once again requires the <li> </li> for each list item.

The difference here, is an Ordered List will appear as a numbered list for every item you have!
Linking to CSS
Well we've covered some of the real basics of HTML, but all we have now is the ability to make some bland text, different sizes and into lists. What if we want to give our page some style? How do we get the HTML and CSS to communicate with each other? Well this section will cover that, progressing everything nicely onto our next page which is the CSS Fundamentals!
Attribute Definition Example
<link rel="stylesheet" href=" "> So the attribute may look scary, but it's really anymore complicated than our anchor tags. You should always place your link within the head of your document with the metadata.

The rel=stylesheet is stating the type of link we are declaring, rel shortened for relevance. And we are telling the link it's a stylesheet.

And you have already used the href" ", which we will just link the location of our stylesheet!
