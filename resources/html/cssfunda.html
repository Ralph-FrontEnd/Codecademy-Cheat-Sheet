What is CSS?

Cascading Style Sheets is one of the main ways HTML gets its design. There is two ways of implementing CSS into your HTML, the first way is inline styling which directly puts your CSS code in your HTML file, but, this can get pretty messy! The more efficient and tidier way is via the CSS ruleset, which takes place within its own file!

Pseudo Class
A Pseudo class is something that can be attached to a selector, a way of modifying the selector depending on what state the selector is in. Below are just a few examples of the pseudo classes you can use.
Value Description Example
:hover The :hover Pseudo Class will activate when we mouse over something (hovering). This is commonly seen when you're hovering over a button/link and the text and background colors change indicating it's clickable!

The example on the right, when you hover over a class named btt, the cursor will become a pointer and the background of the anchor tag element will become a dark blue color!

CSS Selector Hierarchy
CSS has a built in hierarchy, the way we select attributes will have a huge impact on how it effects the design styling on our HTML. This section will cover some of the basic hierarchy seen within CSS, and will allow you to see that the most powerful selectors will overwrite the lesser powerful ones.
Most Power
Selector Description
Specificty The more specific you are in regards to what you're targetting, will always have the most power. This is because you're taking either the class or ID and then selecting a child within that parents class.
For example:
.myClass li
The above is using a class which would be on the UL/OL and the list item, making it more specific to that exact section.
ID's The ID selector is the most powerful standalone selector, in an ideal world you'd only use a specific ID once within your HTML code.
Classes Classes are slightly weaker than ID's, but still have a good amount of power over the below items. The great positive side of classes is they're designed to be used multiple times throughout our HTML, ideal for repetition.
Attributes Our generic page attributes such as <p> or <h1> are fairly low within the hierarchy, but are still useful to use if you're styling them as a group across your pages.
Gloabl/Universal (*) The asertisk * acts as a global modifier, and is great for setting the main font you'll use or overall font size etc. Things that'll be default across your page and appear a LOT!
Least Power

Colors
Adding color to your website is one of the ways of making it yours, having your color scheme of choice! There are a few different ways you can get color onto your page, you can use color: to set the text of the actual item, and background-color: to change the color of the background. See more below!
Selector Description Example
Hexadecimal #ffffff
This is the most common way you'll see colors represented, through their Hex. These can be a mixture of numbers and letters, consisting of either 6 or 3 values behind the #. You will only get 3 values if the values can be repeated twice, so our #ffffff can be shortened to #fff and we'll get the same color.

Our example on the right hand side will change the header class color to a dark blue, and our background-color is transparent.
RGB Colors In RGB we have 3 color options that are then blended together, with the first color representing red, second green and third blue (RGB). Each of the 3 sections on RGB can be anything between 0 and 255 giving you the exact same range as Hex with your color.

The example on the right will change all our pargraphs to a light grey color.
Hue Saturation & Lightness Our third and final way of calling color is one of the harder ways to nail down the exact color you want, although the syntax is similar to RGB, it acts a little differently.
The first number represents the hue, it refers to the angle on a color wheel, with red being 0 degrees, green 120 degrees and blue 240 degrees, back to red again at 360.
Saturation refers to the intensity of the color, 0% would become grayer and 100% the color becomes more vibrant.
And finally the lightness is for how light or dark the color is, so 50% is normal, and then imagine sliding a dimmer switch in either direction.

The example on the right handside will set our paragraph color to a vibrant purple!
