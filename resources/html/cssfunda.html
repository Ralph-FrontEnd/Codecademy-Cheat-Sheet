What is CSS?

Cascading Style Sheets is one of the main ways HTML gets its design. There is two ways of implementing CSS into your HTML, the first way is inline styling which directly puts your CSS code in your HTML file, but, this can get pretty messy! The more efficient and tidier way is via the CSS ruleset, which takes place within its own file!

Padding
The best way I can think of describing padding is like this! Imagine you're standing in a box, and this box by default is touching you on all sides, it is designed to fit to your needed dimensions. However, we can give you a little more room by using padding! And the way padding works is you choose how much space you want on each of the 4 sides to let yourself breathe a little easier, the walls are still there just a little further away!

padding Calling each area that needs padding can be time consuming and pain staking writing it over and over, so what about shorthand version?

Code Example Explanation
So now things are getting a little more interesting! We have 4 sets of numbers but no specific areas, how does our CSS code know? Well it's built in!

Our first value = Top

Our virst value = Top & Bottom
Our second value = Right & Left

Pseudo Class
A Pseudo class is something that can be attached to a selector, a way of modifying the selector depending on what state the selector is in. Below are just a few examples of the pseudo classes you can use.
Value Description Example
:hover The :hover Pseudo Class will activate when we mouse over something (hovering). This is commonly seen when you're hovering over a button/link and the text and background colors change indicating it's clickable!

The example on the right, when you hover over a class named btt, the cursor will become a pointer and the background of the anchor tag element will become a dark blue color!
:active If we are sticking to a button, the active pseudo class can be extremely useful too. The term active means it'll change what's happening whilst you're doing it, in terms of a button whilst you're clicking.

The example on the right, the paddings will shrink to make it look like the button is getting smaller when clicked on due to the button:active being 2 pixels less on padding.
:visited The final pseudo class we'll look at is the :visited value. You'll have noticed when clicking a link on a browser, within Microsoft word, pretty much anywhere. Unless someone has modified this value, the default link goes from being a blue color, to a purple color once you've clicked that link.

The example on the right, this would actually remove all styling that the default :visited value has built in.
