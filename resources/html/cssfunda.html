What is CSS?

Cascading Style Sheets is one of the main ways HTML gets its design. There is two ways of implementing CSS into your HTML, the first way is inline styling which directly puts your CSS code in your HTML file, but, this can get pretty messy! The more efficient and tidier way is via the CSS ruleset, which takes place within its own file!

Padding
Sometimes the items on our pages can be a little too surrounded, almost like they need breathing room. One of the ways we can do this is padding, the easiest way to describe padding is imagine a box. And inside the box we have another box, you can add padding to the inside box which gives you a little more space between the two boxes. But really there is only one box, we're just giving everything more space on the inside!
